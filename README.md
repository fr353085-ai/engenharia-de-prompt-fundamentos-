Prompt = O que é LLM
 Inteligência Artificial (Large Language Model) 
Um Large Language Model (Grande Modelo de Linguagem) é um tipo de inteligência artificial. Ele é treinado com grandes quantidades de dados textuais para entender e criar linguagem humana. 

Prompt =Atue como um professor experiente. Explique LLM para um iniciante com o máximo de 2 parágrafos
Imagine um LLM (Grande Modelo de Linguagem) como um assistente que leu quase todos os livros, artigos e conversas já escritos na internet. Ele não "pensa" como nós, mas funciona como um mestre da probabilidade: ao receber uma frase, ele analisa bilhões de padrões para prever, palavra por palavra, qual é a resposta mais lógica e natural para completar aquele pensamento. É como um corretor automático de celular, só que infinitamente mais sofisticado e capaz de entender contextos complexos.
Esses modelos são redes neurais amplas, treinadas para traduzir, resumir, criar códigos e manter diálogos. Eles convertem a linguagem humana em números e cálculos matemáticos. Isso permite identificar relações entre ideias. Ferramentas como o ChatGPT ou o Gemini podem responder perguntas de forma coerente. Em resumo, um LLM é uma tecnologia que aprendeu a estrutura da comunicação. Ele atua como um simulador de inteligência e linguagem.
